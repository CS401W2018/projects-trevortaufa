  1. What is normal flow in HTML?

    Normal flow is the default layout behavior of elements in HTML. It follows these rules:

    • Block elements (like div, p, h1 ) stack vertically from top to bottom.
    • Inline elements (like span, a, strong) flow horizontally inside their containing block.
    • Elements are positioned based on the document structure unless altered by CSS properties like float, position or display

  2. What does float do? Why do you need to clear the float?

    Float allows elements to be taken out of the normal flow and positioned to the left or right of their container. This is often used for text wrapping around images or creating grid-based layouts before Flexbox/Grid.

    Clearing float is necessary because floating elements can cause their parent container to collapse (since the container no longer recognizes their height)

  3. What is absolute, fixed and relative?

    • Relative: Positions an element relative to its normal position (it moves but it still occupies space)
    • Absolute: Positions an element relative to the nearest positioned positioned ancestor (or the viewport if none exist). It is removed from normal flow.
    • Fixed: Positions an element relative to the viewport, meaning it stays in place even when scrolling.
