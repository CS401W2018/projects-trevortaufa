  1. What is JavaScript? How did it get its name?

    2. JavaScript is a versatile programming language primarily used for web development. It was originally called Mocha, then LiveScript, before being renamed to JavaScript by Netscape to capitalize on Java's popularity, ene though the two languages are not related.

  2. What are the two ways you can run JavaScript?

    3. You can run JavaScript in two main ways:

    • Inside a web browser (where it controls webpage behavior, like animations and interactivity)
    • on a server using Node.js, which allows JavaScript to handle backend logic, databases and APIs.

  3. What was the most popular front-end js library?

    4. JQuery was the most popular front-end JavaScript library, simplifying DOM manipulation, animations, and AJAX requests. It was widely used in the 2000s and early 2010s because is amde JavaScript mroe accessible and worked across different browsers.

  4. What is the most popular front-end js library today?

    5. React, developed by Facebook (now Meta), is the most popular front-end JavaScript library today. It uses a component-based architecture and a virtual DOM, making it efficient for building interactive and scalable web applications.
