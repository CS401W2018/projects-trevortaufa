Engagement - Form Processing with AJAX

  1. What is AJAX?

    AJAX stands for Asynchronous JavaScript and XML. It's a technique used in web development to create dynamic, interactive web applications. AJAX allows a web page to update parts of the page without reloading the entire page. This improves the user experience by making websites faster and more responsive:

    Here's how it works:

    1. Asychronous: AJAX allows communication with the server in the background without interrupting the user's interaction with the page. When a request is made, the rest of the page can still be interacted with.
    2. JavaScript: JavaScript is used to send the request to the server and handle the server's repsonse.
    3. XML (or JSON): Originally, the data exchanged between the client and server was in XML format, but now JSON (JavaScript Object Notation) is mroe commonlyused due to its simplicity and ease of use.
    4. HTTP Requests: AJAX uses the XMLHttpRequest (XHR) object (or the newer Fetch API) to send requests to the server and retrieve data asychronously.

  2. Why do you put your JS Code in a separate file?

    Keep your HTML separate to your JS Code, keeping things cleaner.
