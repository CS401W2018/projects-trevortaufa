  1. Why is the viewport meta tag important?

    2. The viewport meta tag is important for ensuring a website is displayed correctly on different devices, especially on mobile screens. It allows developers to control how a web page is scaled and displayed on various screen sizes.

  2. How can you adjust the CSS for printing?

    3. Steps to adjust CSS for printing:

    1. Create another CSS file for the page
    2. Copy and paste the existing CSS code to the printing CSS
    3. Link the printing CSS to the HTML
    4. Add the label 'media="screen'" to the existing CSS link
    5. Add the label 'media="print"' to the Printing CSS link
    6. Adjust CSS to desired look of the printable page.

  3. When would you want to ajdust the CSS for printing?

    4. You'd want to adjust the CSS for printing when you need to optimize the layout, readability, and appearance of your web page for printed documents. Web pages are designed for screens, but printed, they can look cluttered, waste ink, or lose important information. Using print-specific CSSensures a cleaner, more professional printed output.
