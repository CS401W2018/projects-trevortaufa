Engagement

Answer the following questions:

  1. List out the different ways to select an item.

    • Tag Name selector: Selects all elements of a specific tag.
    • ID Selector: Selects a unique element with a specific "id".
    • Class Selectors:Selects multiple elements with a shared "class"
    • Child Selectors (>):Selects direct children of an element
    • Descendant Selector (space):Selects all nested elements with a parent.

  2. What is the difference between an ID and a Class?

    • ID
      • Should be unique per page
      • "#idname"
      • Higher Specificity
      • Ideal for unique elements
    • Class
      • Can be used multiple times
      • ".classanme"
      • Lower specificity
      • Ideal for reusable (e.g. ".button")

  3. What is the difference between a SPAN tag and a DIV tag?

    • SPAN
      • Inline
      • Used for styling part of a text within a line
    • DIV
      • Block
      • Used for structuring layouts
      • "div style="background-color:gray"

  4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

    • DIV
      • No specific meaning
      • Generic container
      • Less meaning for SEO
    • HEADER, MAIN, or FOOTER
      • Represents specific sections of a page
      • Used for structural page elements
      • More meaningful for SEO

  5. What are the different ways to select a color?

    • Named Color
    • RGB
    • Hex Codes
    • short hexcode
    • HSV (Hue Saturation Value)

Definitions:

CSS Selectors
CSS Selectors define the pattern used to select elements for styling. They allow you to apply styles the specific elements based on their attitudes, hierarchy, or state.
Tag Name
A tag name refers to the HTML element itself, such as "div", "p", "h1", etc. It can be used as a CSS selector to style all instances of that tag.
ID
An ID is a unique identifier for an HTML element, assigned using the "id" attribute. It is referenced in CSS using a "#". An ID should be unique within a page
Class
A class in an attribute that can be assigned to multiple HTML elements to group them for styling. It is referenced in CSS using "." (dot).
Children
In HTML, children often refer to the elements nested inside a parent element. CSS can target children using ">" (direct child) or space (descendant) selector.
The Span Tag
The "span" tag is an inline container used for applying styles or scripting to a specific section of text within a block of content. Unlike "div", it does not create a new line.
