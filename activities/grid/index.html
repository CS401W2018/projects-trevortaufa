  2. What is the main difference between CSS Grid and CSS Flex?

    CSS Grid:

    • Grid is split up into columns and rows.
    • Strict with children size and place.

    CSS Flex

    • Grid is 1 dimensional
    • Grid will tell the child where to go but is flexible with size.
