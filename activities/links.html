2. Absolute links vs Relative links

Absolute links

Absolute links are links that direct the user to another website on the web. The absolute link is made up of 3 components:

  • Protocol (https://)
  • Domain (example.com)
  • Path (/example/path/)

Relative links

Relative links are links within the website/root folder. Like the name implies, it uses family relative names to help describe locations of the links. Here are some example of the terminology:

  • Root Folder / Parent
    • File.html
    • File2.html (Sibling file)
    • Folder (Child)
      • filename.html (Niece/Nephew)
  • Another Folder (Uncle/Aunty)
    • filename1.html

4. 3 Rules of Relative links:

  1. Same folder (sibling):
    • filename.html
  2. Child folder:
    • folder/filename.html
  3. Up Folder (Aunt/Uncle)
    • ../filename.html
    • "../" goes up a folder
  4. *EXTRA RULE: Start at root on website:
    • "/____"
