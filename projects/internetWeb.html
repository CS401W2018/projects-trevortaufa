Internet

The word derives from the word Internetworking, which is what the internet is: a global network of interconnected computers. Like highways are to cars, the internet is a network for information to be exchanged and communicated to many other computers.

Web

The World Wide Web, often shortened to the web, is a network of information that can be accessed via the internet, as if the internet was is the road and highways and the web is the destinations people are trying to get to. Like the roads lead to homes and stores, the internet allows us to access various information and sites.

How are the Internet and Web different?

As explained earlier, the internet is the medium by which computers communicate to each other and the web is a network if stored data that can be accessed.

ASCII and Unicode

ASCII and Unicode are both character encoding standards that represent text and characters on computers.

ASCII

ASCII (American Standard Code for Information Interchange) is a system that uses up to 8 bits to represent each character. Because a byte is only limited to 8 bits, the limit of characters in the ASCII system is 256. It was the first major character encoding standard for data processing.

Unicode

Unicode is a unviersal character encoding standard that represents characters from many languages and writing systems. The difference between Unicode and ASCII is that Unicode uses more than one byte to represent each character, allowing it to represent more characters. Not only does it include characters from other languages but also represents emojis, symbols and characters from different scripts.

What is UTF-8?

UTF-8 (Unicode Transformation Format 8) is a character encoding system that represents characters using one to four bytes and is the most common character encoding on the web.

URL

URL (Uniform Resource Locator) is the address of a web page. As the name implies, it is a universal Locator that can be accessed from any browser that has access the the interent and the web.

Domain Name

A domain name is a user friendly, text-based address that identifies a website on the internet. Unlike IP addresses, domain names are designed to be easily remembered by humans, like "google.com" instead of "172.217.160.142".

HTTP

HTTP stands for "Hypertext Transfer Protocol". It's the computer communication protocol used for most communication on the world wide web. When you visit a website, your browser sends a HTTP request to the web server, which responds with a HTTP response.

HTTPS

HTTPS is the same as HTTP, but with an S, meaning secure. As the name suggests, HTTPS is a more secure version as the data is encrypted before it is transferred.

IP Address

An IP (Internet Protocol) address is the unique identifying number assigned to every device connected to the internet.

DNS

DNS stands for Domain Name System. It is like the phonebook of the internet. Humans access information online through domain names, web browsers interact through IP addresses and DNS translates domain names to IP addresses so broweser can load internet resources. DNS eliminates the need for humans to memorize IP addresses like google was mentioned earlier.

W3C Recommendations

W3C (World Wide Web Consortium) Recommendations refers to a set of standards or guidelines published by the W3C to define how web technologies should function to promote consistency and interoperabilty across platforms and browsers.
